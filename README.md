# Gestion_Projets_Calcul
Application Python pour collecter les donnée des projet de calculs.

## Prérequis
la version de python installé sur la machine: Python 3.13.0

## Installation des dépendances
depuis Gestion_Projets_Calcul lance la cmd : pip install -r requirements.txt

## Lancer le projet

### Interface Terminal (originale)
depuis Gestion_Projets_Calcul lance la cmd : py .\main.py

### Interface Graphique Tkinter (nouvelle)
depuis Gestion_Projets_Calcul lance la cmd : py .\front_end_tkinter.py

## Fonctionnalités

### Interface Terminal
- Saisie interactive dans le terminal
- Gestion basique des projets

### Interface Graphique Tkinter
- Interface moderne avec fenêtres
- Liste des projets avec détails
- Ajout, modification, suppression de projets
- Duplication de projets
- Saisie intuitive avec listes déroulantes et cases à cocher
- Validation des données
- Sauvegarde automatique en JSON
